中国225万平方公里的热带疆域在哪里？

原创 张雨晨 中国国家地理BOOK 2020年02月07日 19:17

▲ 沙滩和大海不是判断热带的标志，左边的植物才是。图/视觉中国

中国的热带，可不止有海南哦！

一到冬天，北方人嫌天气冷，想去海南过冬，南方人嫌没暖气，也想去海南过冬。位于热带的海南，太阳终年可以直射大地，这里并没有严格意义上的冬季。 除了海南，中国还有哪些热带地区呢？让我们一起来看一看吧！

墨脱：世界最北的热带

墨脱位于西藏东南，喜马拉雅山脉东段南麓，它所在的纬度，是北纬29度。国际上公认的热带范围，在南北回归线（23°26′）之间，所以，从理论上讲这里并不是热带。

但是，西藏东南是一片神奇壮丽的土地。汹涌的雅鲁藏布江在南迦巴瓦（海拔7782米）和拉加白垒（海拔7294米）两座大雪山中间，硬生生在喜马拉雅山脉上劈开的一个大口子，这就是雅鲁藏布大峡谷。印度洋的暖湿水汽，以这条峡谷为通道，源源不断来到原本寒冷干燥的青藏高原。这条大峡谷输送水汽量的强度，与夏季自长江以南向北输送的水汽强度都相当。
)

▲ 北上的暖湿气流在这形成了大量的降水，造就了此地丰富的森林植被。 图/图虫·创意

来自印度洋的湿和暖，在雅鲁藏布大峡谷底部，造就了地球上最北的热带季雨林。和西藏似乎完全不搭边的香蕉、芭蕉、菠萝等热带水果，在这里都可以见到。
)

▲ 门巴族小朋友抱着一串香蕉。香蕉是当地人常吃的食物之一。 图/图虫·创意

湍急的雅鲁藏布江日夜奔腾，将河底的泥沙和石头淘去了许多，这让雅鲁藏布大峡谷成为了世界上最深的峡谷，墨脱，也因此被称为了“世界屋脊的低谷”。从海拔200米的墨脱县巴昔卡村，到海拔7782米的南迦巴瓦峰顶，超过7500米的落差，造就了中国唯一的具有完整自然垂直带谱的山地。9个垂直分布的自然带，几乎集中了从海南到北极的各种植被，堪比一个浓缩的地球。
)

▲ 南迦巴瓦峰。图/《发现西藏》，摄影/姚赫

在墨脱，人们可以开着车，在半天的时间内，感受到一年四季的变换。墨脱的大门，是海拔3700米的嘎隆拉隧道，这里眼及之处尽是极地风光。从嘎隆拉隧道到海拔1100米的墨脱城，再下降到海拔800米的背崩，不到110千米的盘山公路，就完成了从冬到夏、从极北到热带的跨越式体验。
)

▲ 墨脱的原始森林。摄影/刘志强

印度洋的水汽，不仅塑造了墨脱的热带环境，还为这里海拔7782米的南迦巴瓦峰“美了容”。当暖湿气流撞上高山之巅，就形成了大量的海洋性冰川。它们的积累和消融都很快，流动性强，对山体的侵蚀、搬运能力强，山顶周围的岩石往往被“刨”走，只剩一个非常尖的山峰，两壁十分陡峭。
)

▲ 月亮挂在白雪皑皑的南迦巴瓦峰上。图/Sino Images

于是，在《中国国家地理》杂志举办的“选美中国”活动中，南迦巴瓦峰一举摘得了最美山峰第一名的桂冠。

西双版纳：“野生动植物王国”

广东、广西、云南和西藏四省、自治区的南部边缘，其实都迈入了热带的界线，但是，这里大部分的热带生态系统，都在长期的农业生产中，被破坏殆尽。农田，灌木、种植林取代了原生的雨林，只有在险峻陡峭、人迹罕至的山地，才残存着几分热带的模样。

让人欣慰的是，西双版纳还在！她横卧在云南省的最南端，保存着中国最大的一片热带雨林和热带季雨林。
)

▲ 西双版纳的热带森林。图/视觉中国

西双版纳的地形，东、北、西三面高，挡住了由北而来的冷空气，夏季来自印度洋孟加拉湾的南亚季风，裹挟着水汽，从南部的多个缺口不断涌入，让这里雨水充沛；冬季，在副热带高压带的控制下，干暖晴朗的天气，吸引着全国游客的目光。 繁茂的热带森林，涵养着水源。澜沧江经由西双版纳走出国境，成为中南半岛上大名鼎鼎的湄公河，罗棱江、南腊河、南阿河、流沙河，20多条支流像毛细血管一样滋养着这片大地，它们共同编织成了西双版纳碧水清流的水源网络。
)

▲ 蜿蜒的澜沧江。图/视觉中国

在暖湿水汽的滋养下，西双版纳一跃成为了“植物王国王冠上的绿宝石”，迄今为止，在这里发现的高等植物多达5000多种，占到全国高等植物总数的六分之一。在西双版纳东、西、北三个方向高山的保护下，许多诞生于第三纪（距今6500万年至300万年，随后就是今天所处的第四纪）的古老植物，如桫椤、云南苏铁，得以在冰期中幸存，一直延续至今。
)

▲ 相比起古老植物，更受游客关注的，还要数好吃的水果。图/视觉中国

只有绿意，未免太过单调，西双版纳的动物资源也毫不逊色。这里是2000多种动物的家，其中，最为人津津乐道的是目前世界上最大的陆栖哺乳动物——亚洲象。在中国境内，如果想看到这种憨态可掬，同时又破坏力极强的野生动物，只能来西双版纳。
)

▲ 亚洲象。图/视觉中国

图片 海南与南海：到天涯海角去 说完了墨脱和西双版纳，终于轮到我们最熟悉的海南出场了！椰树、阳光、沙滩、碧海，这里是中国最知名的度假胜地。当北国千里冰封之时，这里依然温暖如夏，让人心醉。

在《中国国家地理》评选出的中国八大最美沙滩中，海南三亚亚龙湾的得票率高居榜首。这里的沙滩，长度超过7千米，比处于同一纬度上著名的夏威夷威基基海滩，要长出4倍多。整片沙滩，在阳光的照射下，散发着银白色的光芒。
)

▲ 亚龙湾的银沙滩。图/视觉中国

亚龙湾沙滩，仿佛是过过筛子一样，质地异常细腻，踩上去柔软得就像踩在面团上一般，走过时留下的“吱吱”声，带来踏雪般的喜悦。这片沙滩好到甚至“惊动了中央”，2008年北京奥运会的“沙排”比赛的场地用沙，都是从这里运过去的。 为什么亚龙湾的沙滩如此之好？要想找到原因，还需先把海南这两个字需要掉个个儿，没错，就是南海。亚龙湾的沙滩，来源于南海中最为繁茂、绚丽的“丛林”——珊瑚礁。白色的沙子，不是石头，而是珊瑚和贝壳经过打磨、风化后形成细小颗粒。
)

▲ 亚龙湾沙滩特写，不过，沙滩上的珊瑚可能是摆拍。图/视觉中国

珊瑚，就像是海底的森林。一种微小的腔肠动物——珊瑚虫，用自己的身体，在海底搭建了五彩斑斓的珊瑚丛林，鱼类、软体动物，就如同生活在丛林中的鸟兽。 我们能够经常看到的珊瑚，其实活得特别“矫情”。它们只会选择亚热带和热带安家，因为水温必须全年保持在20-30摄氏度之间；它们家中的海水必须洁净，透明度高，所以在河流入海口等水质浑浊的地方，不会有它们的身影；在水深70米以下的海底，因为光照不足，它们也从不光顾。
)

▲ 美丽的珊瑚。图/视觉中国

南海上的西沙群岛，是我国沿海珊瑚礁生态系统的发源地。这里夏季水温在29度左右，冬季也不会降到24度以下，光照充足，是珊瑚生长的理想场所，珊瑚礁总面积预计超过了100平方千米。 从空中俯瞰，西沙的岛屿，就像是一块块镶嵌在蓝丝绒上的绿宝石，白色的镶边底座，是沙滩，翠绿的中央，是热带丛林。在西沙群岛的数十个岛屿、环礁之中，除了基岩岛之外，其它都是由珊瑚形成的岛礁，是珊瑚造就了这片珍贵的南国重镇。
)

▲ 中国南海的珊瑚岛屿。图/视觉中国

与茂盛生长的珊瑚相伴的，是丰富的海洋生态系统。西沙群岛有鱼类400余种，这其中就有我们熟悉的金枪鱼、石斑鱼等美味。在西沙，还生活着名贵的“海龟之王”棱皮龟、“海参之王”梅花参，以及能够产出世界级的珍珠——南珠的马氏珠母贝。 西沙群岛远离祖国大陆，人类活动也比较稀少，这里一直清澈的海水和茂盛的珊瑚，是旅游者心向往之的秘境。2014年，当西沙向国人开放旅游后，它就获得了“中国的马尔代夫”的美誉。
)

▲ 西沙群岛岛上的灯塔。图/视觉中国

其实，西沙群岛的玩法，和马尔代夫完全不同：大部分游客仅能登陆全富岛、银屿岛和鸭公岛，而且没法在岛上享受星级酒店（因为根本没有）。南海的美，不仅在于它的景色，更因为这个地名前面的不可或缺的中国二字。 总有一天，我们还会再往南走，前往南沙群岛和曾母暗沙，去探索这些从小就印刻在我们记忆深处的地方。

图片 台湾：宝岛的一抹绿

在台湾省西侧的嘉义县和东侧的花莲县，各立有一块北回归线标志碑，从原则上来说，北回归线以南，都是热带。

▲ 台湾省的北回归线地标。图/视觉中国

其实，台湾省的实际热带区域，已经越过了北回归线。日本暖流（又叫黑潮）在流经台湾岛东海岸时，为这里加了温，从气候条件而论，台湾岛全岛都具有长夏无冬的热带气候特征。 但是，如果你在网络上搜索“台湾岛热带”，你得到的结果往往是：“台湾岛的热带主要分布在台湾南部的恒春半岛、兰屿和绿岛”，或者是“台湾本岛唯一的热带区域——垦丁公园”等等。
)

▲ 垦丁的鹅銮鼻公园。图/视觉中国

垦丁就位于恒春半岛南部，如果坐飞机去垦丁，降落的机场就叫恒春机场。恒春半岛的北缘，纬度大约是北纬22度，说好的全岛热带，都跑到哪里去了呢？ 台湾的热带，和中国广东、广西的热带状况类似，也就是说，这里只有热带之名，却无热带之实，明明是热带的地方，却长满了亚热带的常绿阔叶林。有些学者认为，这背后的原因，是历史上的过度砍伐。
)

▲ 台北大学的椰子树，椰子树是典型的热带植物，台北大学也以椰林大道而闻名。图/视觉中国

1895年，日本占据了台湾，在当年10月颁布的《官有林地取缔规则》中，规定“如无足以证明所有权的地券或其它确据之森林原野，概为官有。”此后，96%的台湾森林，都归了占领当局。在1926年时，台湾全岛还有超过了210万公顷的森林，但到了1942年，森林面积骤减了近90%，只剩下大约17万公顷了。 自1949年开始，为了挽救财政困局，当局也开始砍伐森林。“上世纪50、60年代，我们必须砍伐森林，卖祖宗财才能活下去”，台湾省生态协会理事长杨国祯在回忆起那段历史时，十分无奈。上世纪60、70年代台湾省经济的崛起和蜂拥而起的建设高潮，再次将这里的森林逼入绝境。
)

▲ 台湾省的热带森林。图/视觉中国

1982年9月1日，回过味儿来的人们，终于公告成立了第一座国家公园——垦丁国家公园，来保护这里珍贵的热带海岸林及热带季雨林。虽然在台湾岛上，还零星分布着其它一些热带雨林的踪迹（如台北县的大豹溪热带雨林），但是大片的热带植物，就只剩下垦丁这一片了。所以，网上才会出现“台湾本岛唯一的热带区域”这种误传。
)

▲ 台湾热带森林中用木头铺成的步道。图/视觉中国

不过，即使是误传，也不要紧，更高的知名度，也让珍贵的热带森林得到了更好的保护。垦丁的旅游，已经成为了当地的名片。 对于海南、西双版纳等热带地区来说，情况也是如此。曾经，橡胶业是西双版纳农民的重要经济来源，季雨林被橡胶园无情地分割得七零八落。但随着旅游业逐渐成为支柱，在不久的将来，这些零散的季雨林，一定还会再次相连成片的。
